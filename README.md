# Bot Releases

Este repositório é utilizado exclusivamente para a hospedagem dos instaladores (`.exe`) e arquivos de atualização automática.

## Como Baixar e Instalar:

1. Navegue até a seção **[Releases](../../releases)** (barra lateral direita).
2. Clique na versão mais recente (ex: `v1.0.5` ou `Latest`).
4. Em "Assets", faça o download do arquivo executável:
   - **Nome do arquivo:** `Bot-de-Mensagens-Setup-X.X.X.exe`
5. Execute o arquivo baixado. O software será instalado e aberto automaticamente.

---

## Sistema de Atualização Automática

Este software possui um sistema de *Auto-Update* integrado.
- **Não é necessário baixar novas versões manualmente.**
- Ao abrir o Bot, ele verificará silenciosamente se há uma nova versão neste repositório.
- Se houver, uma notificação aparecerá solicitando a reinicialização para aplicar a atualização.
